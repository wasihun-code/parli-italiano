# Conversation Rules

## Structure
*   **Minimum Count:** 4 scripted, branching conversations per scenario.
*   **Length:** 10 to 20 turns minimum per conversation. They must feel like complete real-world interactions with a clear beginning, middle, and end.
*   **Initiation:** Every conversation MUST begin with the host (or equivalent AI persona) speaking.

## Pedagogical Quality
*   **Level:** Target A1-A2 learner.
*   **Simplicity:** Short, focused sentences. Exactly one concept or instruction per sentence.
*   **Naturalness:** No robotic repetition. Use realistic spoken Italian (e.g., "Sì, sono dentro" instead of repeating the full prompt).
*   **Challenges:** Include realistic learner challenges (misunderstandings, clarification requests, physical environment issues).

## User Interaction
*   **Choices:** Every user response node must contain at least 3 choices (1 correct, 2+ distractors).
*   **Format:** Choices must be full phrases/sentences that the user would say.
*   **Audio:** User reply audio DOES NOT autoplay. It is available manually.

## UI/UX Integration
*   **Immersive Mode:** Conversations run in a fullscreen, distraction-free environment.
*   **Progressive Reveal:** Host text is initially hidden while audio plays. The learner must actively choose to "Show Text".
*   **Translation:** Translations are available only after the Italian text is revealed.
*   **Keyboard Support:** Full keyboard navigation (`1-4`, `Enter`, `Space`, `T`, `Esc`) is required.
